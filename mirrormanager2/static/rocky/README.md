# Rocky Linux Static Files

These static files are sourced from the mirrormanager-rocky repository
(originally at: mirrormanager-rocky/static/rocky/).

They provide Rocky Linux branding including:
- Rocky green color scheme (#10b981)
- Bootstrap 5.3.3 CSS framework (in fedora-bootstrap/)
- Font Awesome and Open Sans fonts (in fonts/)
- Interactive map assets (in map/)
- Favicon and logo images
- Chart.js for statistics visualization

## File Sources

All files in this directory were copied from:
`mirrormanager-rocky/static/rocky/`

## Key Files

- `rocky.css` - Main Rocky Linux stylesheet with green color theme
- `rocky.min.css` - Minified Bootstrap with Rocky customizations
- `fedora-bootstrap/` - Bootstrap 5.3.3 compiled CSS/JS
- `fonts/` - Font Awesome and Open Sans web fonts
- `map/` - OpenLayers and map marker assets
- `mirrormanager-logo.svg` - Rocky-branded MirrorManager logo
- `favicon.ico`, `favicon.png` - Rocky Linux favicon
